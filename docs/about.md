# 项目信息

## 客户
杭实集团（杭州市实业投资集团有限公司）

## 项目概述
为杭实集团构建集团级统一门户系统，作为世界500强企业的数字化升级项目，整合旗下116家控股企业、34家直接参控股企业的信息与服务，提供统一的入口和管理平台。

## 核心需求
- 集团统一信息门户，整合各子公司信息
- 统一身份认证与权限管理
- 数据可视化与决策支持
- 移动端支持

## 行业
- 国有投资集团
- 产业投资与管理
